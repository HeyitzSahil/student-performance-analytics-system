# Student Performance Analytics System (CLI)

A command-line based application built using Python and MySQL to manage and analyze student academic performance.

## Features

* Add, update, and delete student records
* View detailed report cards
* Subject-wise toppers
* Top 3 performers
* Weak student detection by subject
* Modular code structure (CRUD + Analytics)

## Tech Stack

* Python
* MySQL
* MySQL Connector

## Project Structure

* db.py → Database connection
* crud.py → CRUD operations
* analytics.py → Performance insights
* main.py → CLI interface

## Future Improvements

* School & class hierarchy
* Stream selection (Science, Commerce, Arts)
* Advanced analytics dashboard
* AI-based performance suggestions

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
